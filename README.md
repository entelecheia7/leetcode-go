# Golang multi-solutions for leetcode
每道题目均有最优算法。  
**AC 不是终点，超越 100% 耗时才是。**  
部分题目的解法在可读性和简洁性上进行了权衡，稍稍牺牲了简洁性。  
BFS&DFS、回溯、动态规划的部分题目存在多种解法，目前我选择其中一种解法进行归类。如果我想到更好的办法再做修改。
